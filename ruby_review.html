One of the code solutions that I was proud of was the regular expressions challenge. The regular expressions challenges can be very difficult but if you take a step back, work on the solution in rubular www.rubular.com, and make sure that your solution also works in your editor and your console you'll be able to come up with a very clear and concise solution. I tried to make my solutions as dry and concise as possible and believe that I did. Looking back there are still somethings I could do to tweak my solution but I feel that it was an almost perfect solution to a relatively complicated problem. I followed best practices for readable code, for pseudocode it wasn't perfect but the pseudocode was very clear.
Determine whether a string contains a Social Security number

    def has_ssn?(string)
      return true if string.match(/\d{3}-\d{2}-\d{4}/)
        else 
      false
    end


Return the Social Security number from a string.

    def grab_ssn(string)
      string.match(/\d{3}-\d{2}-\d{4}/).to_s if string.match(/\d{3}-\d{2}-\d{4}/)
    end

Return all of the Social Security numbers from a string.

    def grab_all_ssns(string)
      string.scan(/\d{3}-\d{2}-\d{4}/)
    end

Obfuscate all of the Social Security numbers in a string.Example: XXX-XX-4430.

    def hide_all_ssns(string)
      string.gsub(/(\d{3})-(\d{2})-(\d{4})/,'XXX-XX-\3') 
    end

Ensure all of the Social Security numbers use dashes for delimiters.Example: 480.01.4430 and 480014430 would both be 480-01-4430.

    def format_ssns(string)
      string.gsub(/(\d{3})[^\d]?(\d{2})[^\d]?(\d{4})/,'\1-\2-\3') 
    end
